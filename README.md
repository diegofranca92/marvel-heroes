### :computer: Como rodar o projeto
```bash
# entre na pasta do projeto
cd marvel-heroes

# rode esse comando pra instalar as dependências do projeto
yarn install
npm install

# rode esse comando pra iniciar a aplicação, e já pode abrir ela no browser
yarn dev
npm run dev

```
